# Logic Levels

Logic Levels is a text-based puzzle game where players solve short logic challenges across multiple levels in the terminal. The game focuses on clear feedback, readable code, and beginner‑friendly logic puzzles.

## Game overview

- Multiple levels of logic puzzles (numbers, patterns, counting, true/false)
- Three attempts per level with contextual hints
- Instant feedback after each answer
- Simple, well‑commented Python code designed for learning and iteration

## How to play

1. Run the game from your terminal.
2. Read the level description and question carefully.
3. Type your answer and press Enter.
4. Use the hints to adjust your reasoning if you get an answer wrong.
5. Clear all levels to win the game.

Each level focuses on a different type of reasoning, such as:
- Even/odd and divisibility rules
- Pattern recognition in sequences
- Counting and observation skills
- Conditional (“if…then…”) logic
